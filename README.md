# 🤖 MindAgent - AI Araştırma Platformu

MindAgent, Groq ve Tavily API'lerini kullanarak internet üzerinden derinlemesine araştırmalar yapan ve profesyonel raporlar sunan bir yapay zeka asistanıdır.

## 🚀 Özellikler
- **Hızlı Analiz:** Groq LPU altyapısı ile saniyeler içinde sonuç.
- **Canlı Arama:** Tavily API ile güncel internet verilerine erişim.
- **Dashboard:** Geçmiş raporları görüntüleme ve yönetme.
- **Kullanıcı Dostu Arayüz:** Modern ve karanlık tema tasarımı.

## 🛠️ Kurulum
1. `npm install` ile bağımlılıkları yükleyin.
2. `.env` dosyasını oluşturun ve API anahtarlarınızı ekleyin.
3. `npm start` ile server'ı başlatın.
